# Code of Conduct

We expect everyone participating in this project to be respectful and constructive.

## Our standards

- Be kind and patient.
- Assume good intent.
- Keep discussions focused on the technical work.
- No harassment, hate speech, or personal attacks.

## Reporting

If you experience or witness unacceptable behavior, please open an issue in this repository with as much context as you can.

## Enforcement

Project maintainers may remove, edit, or reject contributions and may ban users who violate this Code of Conduct.
