# Chrome Web Store listing (draft)

## Name
ChatGPT Chat Exporter

## Short description
Export ChatGPT chats to JSON files named by chat title.

## Description
Export your ChatGPT chat history from the web UI to local JSON files.

How it works:
- Open ChatGPT on the web.
- Open the extension popup.
- Export the current chat or export chats from the sidebar list.

Notes:
- Exports are created locally in your browser as JSON downloads.
- You may need to allow multiple downloads in Chrome for chatgpt.com.
- This extension is not affiliated with OpenAI.
